## 📥 玩家下載與安裝教學

如果你是不懂程式碼的普通玩家，請直接跟著以下步驟下載：

1. **尋找下載區**：在目前這個網頁的**最右側**，你會看到一個區塊叫做 **[Releases]**。
2. **點擊版本號**：點選上面寫著 `Latest` 或最新版本號（例如 `v1.0.0`）的連結。
3. **下載檔案**：在該頁面的最下方會有一個 **Assets** 區塊，點開後，點擊 **`楓星分寶計算機 Setup 1.0.0.exe`** 即可開始下載。

### 🚨 首次開啟注意事項
由於本軟體為作者獨立開發，尚未購買微軟數位簽章，Windows 首次開啟時可能會跳出藍色的防毒警告：
* 請點擊左側文字的 **「其他資訊」 (More info)**。
* 接著點擊右下角的 **「仍要執行」 (Run anyway)** 就可以順利開啟囉！

## 🎮 核心功能介紹

* **💰 精準扣稅**：輸入拍賣場的標價，系統自動幫你扣除 3% 手續費。
* **✂️ 浮動剪刀成本**：依照當天里程商店的匯率，自動精算白金剪刀的代墊成本。
* **👻 搞丟道具處理**：支援「自行吸收原價賠償」與「當作沒打到共同承擔」兩種模式，手殘黨的救星。
* **🧾 終極結算清單**：自動結算「誰該吐多少錢出來」跟「誰該拿多少錢」，照著清單交易，1 塊錢都不會算錯！
* **🧹 一鍵重置**：打完下一場王？按下一鍵清除，馬上開始新的計算。
